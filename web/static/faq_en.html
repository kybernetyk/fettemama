fefemama 3.0

This is the newest version of my Blog. It started, years ago, with Wordpress 1.x and went on to a custom, C99 based blogging system. But that became boring. And I decided to make something totally different.

So I took python and mongodb and built a blog. Using HTTP all over again sounded so mainstream and I went instead with ... Telnet.

In the making I realized Python was not my thing and so I took up Go (golang) and remade it.

You can take a look at the telnet frontend by telnetting to port 1337:

% telnet fettemama.org 1337

Connect, type help and explore. (Most blog posts are in German though.)
You can end the session by typing quit.

If you got curious about the blogging software just go to GitHub and take a look at it.

Contact me via email if you've got something to say. Or leave a comment on the blog.

Seriously why telnet?
Because I hate captchas and this is a nice way to let my readers comment without using any annoying anti-spam stuff. Spam bots blatantly ignore telnet.
